Modern Calculator - Installation Instructions

IMPORTANT: Before using the calculator
==================================
1. Install Python (Required):
   - Go to https://www.python.org/downloads/
   - Click "Download Python" (get the latest version)
   - Run the installer
   - IMPORTANT: Check the box "Add Python to PATH" during installation
   - Click "Install Now"

Installation Steps:
================
1. Right-click the zip file and select "Extract All..."
2. Choose a location to extract the files
3. Open the extracted folder
4. Double-click "Launch Calculator.vbs"

If You See Any Errors:
===================
1. Make sure you've installed Python:
   - Open Command Prompt (press Windows + R, type "cmd", press Enter)
   - Type "python --version" and press Enter
   - If you see "Python is not recognized", you need to install Python

2. Make sure files are extracted:
   - Don't run directly from the zip file
   - Extract ALL files to a folder first
   - Make sure calculator.py and Launch Calculator.vbs are in the same folder

3. Try running manually:
   - Open Command Prompt
   - Navigate to the folder with the files (use "cd" command)
   - Type "python calculator.py" and press Enter

Still having problems?
====================
1. Try reinstalling Python:
   - Go to Control Panel
   - Uninstall Python
   - Download and install Python again
   - Make sure to check "Add Python to PATH"
